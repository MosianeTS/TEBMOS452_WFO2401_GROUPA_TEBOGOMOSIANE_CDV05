Task Management Application

Agile Task Management Application

A task management application which allows users to organise their tasks efficiently. Tasks are organised into different boards and further categorised into three different statutes which are 'todo', 'doing' and 'done'.

The application allows users to add a task, update task information and delete a task. The tasks are displayed on to the user interface and stored on local storage. Users can also toggle between a light and dark mode theme. A user can also toggle the appearance of a navigation sidebar.
